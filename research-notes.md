# Research: Thai Hospital Document Formats

## Key Document Types and Their Visual Formats

### 1. บัตรประจำตัวผู้ป่วย (Patient ID Card)
- **Format**: Credit-card sized, horizontal layout
- **Elements**: Hospital logo (top-left), Hospital name, Patient photo (right side), HN number, Full name, DOB, Blood type, Allergies, ID number, Barcode/QR
- **Colors**: Hospital brand colors (typically blue/green header stripe)
- **Photo**: Required - passport-style headshot

### 2. ใบสั่งยา (Prescription - Rx)
- **Format**: A5 or half-page, portrait
- **Elements**: Rx symbol (top-left), "ใบสั่งยา" header, Date, Patient name, Case number, Allergy info, Drug list with dosage instructions, Doctor signature, License number
- **Colors**: White background, minimal color, "เฉพาะเภสัชกร" badge (orange/red)
- **Photo**: Not required

### 3. ใบรับรองแพทย์ (Medical Certificate)
- **Format**: A4, portrait, formal document
- **Elements**: Hospital letterhead with logo, "ใบรับรองแพทย์" title, Doctor info (name, license, specialty), Patient info, Examination date, Symptoms/diagnosis, Treatment, Rest days recommendation, Doctor signature
- **Colors**: Hospital brand header, formal black text
- **Photo**: Not required

### 4. ใบส่งต่อ/Referral
- **Format**: A4, portrait
- **Elements**: Sending hospital letterhead, "ใบส่งต่อผู้ป่วย" title, Patient demographics, Diagnosis, Treatment given, Reason for referral, Receiving hospital, Doctor signature
- **Photo**: Not required

### 5. สรุปข้อมูลผู้ป่วย (Patient Summary / IPS)
- **Format**: A4, multi-section
- **Elements**: Hospital logo, Patient demographics with photo, Active conditions, Medications, Allergies, Immunizations, Recent procedures
- **Photo**: Required

## Hospital Branding Elements (Thai Hospitals)
- **Siriraj**: Blue/navy, ศิริราช logo, Mahidol University crest
- **Ramathibodi**: Green/teal, รามาธิบดี logo
- **Bumrungrad**: Blue/gold, modern corporate style
- **Chulalongkorn**: Pink/magenta, จุฬาลงกรณ์ logo

## Design Principles for Digital Credential Display
1. Use hospital brand color as header/accent
2. Show patient photo prominently for identity credentials
3. Use card-like layout (rounded corners, shadow) for ID cards
4. Use document/certificate layout for formal documents
5. Include verification badge (green checkmark) for active credentials
6. Show QR code for verification
7. Add "สำเนา" watermark for previews
